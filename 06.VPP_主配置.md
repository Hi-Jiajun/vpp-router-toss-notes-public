# VPP 主配置

## 1. `/etc/vpp/startup.conf`

```conf
unix {
  nodaemon
  log /var/log/vpp/vpp.log
  full-coredump
  cli-listen /run/vpp/cli.sock
  startup-config /etc/vpp/startup.user.vpp
  gid vpp
}

api-segment {
  gid vpp
}

memory {
  main-heap-size 1G
  main-heap-page-size 1G
}

cpu {
  main-core 1
  corelist-workers 2-4
}

buffers {
  buffers-per-numa 65536
  page-size 1G
}

statseg {
  size 96M
}

plugins {
  plugin default.so { enable }

  plugin dpdk_plugin.so { enable }
  plugin dhcp_plugin.so { enable }
  plugin acl_plugin.so { enable }
  plugin nat44_ei_plugin.so { enable }

  plugin linux_cp_plugin.so { enable }
  plugin linux_nl_plugin.so { enable }
  plugin tap_plugin.so { enable }

  plugin ppp_plugin.so { enable }
  plugin pppoe_plugin.so { enable }
  plugin pppoeclient_plugin.so { enable }
  plugin pppox_plugin.so { enable }

  plugin ping_plugin.so { enable }
}

dpdk {
  no-multi-seg

  dev default {
    num-rx-queues 4
    num-tx-queues 4
    num-rx-desc 1024
    num-tx-desc 1024
  }

  dev 0000:01:00.0 {
    name wan0
  }

  dev 0000:02:00.0 {
    name lan0phy
  }
}
```

## 2. `/etc/vpp/startup.user.vpp`

```bash
exec /etc/vpp/bootstrap.user.vpp
exec /etc/vpp/routes.telegram.vpp
exec /etc/vpp/routes.telegram6.vpp
exec /etc/vpp/custom.routes.vpp
exec /etc/vpp/linux-egress.vpp
```

## 3. `/etc/vpp/bootstrap.user.vpp`

```bash
# MTU
set interface mtu packet 1500 wan0
set interface mtu packet 1500 lan0phy

# Interface up
set interface state wan0 up
set interface state lan0phy up

# LCP
lcp lcp-sync on
lcp create lan0phy host-if lan0

# LAN address
set interface ip address lan0phy 192.168.1.1/24

# PPPoE
create pppoe client sw-if-index 1 host-uniq 1
set pppoe client 0 username <PPPOE_USERNAME> password <PPPOE_PASSWORD> mtu 1492 mru 1492 timeout 10 use-peer-dns add-default-route

# Static route
ip route add 7.0.0.0/8 via 192.168.1.4

# IPv6
dhcp6 client pppox0
dhcp6 pd client pppox0 prefix group hgw
set ip6 address lan0phy prefix group hgw ::1/64
set interface ip address lan0phy fd00:1234:5678:1::1/64

# NAT44 EI
nat44 ei plugin enable sessions 1048576 users 16384
nat44 ei forwarding disable
nat44 ei mss-clamping 1452
nat44 ei add interface address pppox0
set interface nat44 ei in lan0phy out pppox0
```

## 4. 说明

- `wan0` 物理口保持 `1500`
- `lan0phy` 物理口保持 `1500`
- `pppox0` 自动协商并使用 `1492`
- `NAT44 EI` 采用：
  - `lan0phy in`
  - `pppox0 out`
- ULA 和 GUA 同时存在：
  - `fd00:1234:5678:1::1/64`
  - `prefix group hgw ::1/64`
