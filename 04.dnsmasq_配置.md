# dnsmasq 配置

## 1. 主配置文件

文件：

- `/etc/dnsmasq.d/router.conf`

内容：

```conf
log-facility=/var/log/dnsmasq.log
log-async=20

interface=lan0
bind-dynamic
listen-address=192.168.1.1

domain-needed
bogus-priv
no-resolv
strict-order
no-round-robin
cache-size=0
local-service
rebind-localhost-ok
stop-dns-rebind
dhcp-authoritative
log-dhcp

domain=example.internal
local=/example.internal/
expand-hosts
host-record=router.example.internal,192.168.1.1,fd00:1234:5678:1::1

server=192.168.1.250
server=192.168.1.2
server=fd00:1234:5678:1::53
server=fd00:1234:5678:1::54

# IPv4 DHCP
dhcp-range=192.168.1.100,192.168.1.200,255.255.255.0,12h
dhcp-option=option:router,192.168.1.1
dhcp-option=option:dns-server,192.168.1.1
dhcp-option=option:domain-search,example.internal
dhcp-option=option:classless-static-route,7.0.0.0/8,192.168.1.4
dhcp-option=tag:aliyun_google_dns,option:dns-server,223.5.5.5,8.8.8.8

# IPv6: stateful + SLAAC
enable-ra
ra-param=lan0,mtu:1500,30,180
dhcp-range=::100,::1ff,constructor:lan0,slaac,64,12h
dhcp-range=fd00:1234:5678:1::100,fd00:1234:5678:1::1ff,slaac,64,12h

dhcp-broadcast=tag:needs-broadcast
dhcp-ignore-names=tag:dhcp_bogus_hostname
dhcp-leasefile=/var/lib/misc/dnsmasq.leases
```

## 2. IPv4 静态地址

文件：

- `/etc/dnsmasq.d/dnsmasq.staticv4.conf`

内容：

```conf
dhcp-host=02:00:00:00:00:11,192.168.1.201,client-win11,2h
dhcp-host=02:00:00:00:00:12,192.168.1.30,switch-core,2h
dhcp-host=02:00:00:00:00:13,192.168.1.40,media-avr,2h
dhcp-host=02:00:00:00:00:14,192.168.1.50,nas-a,2h
dhcp-host=02:00:00:00:00:15,192.168.1.20,docker-host,2h
dhcp-host=02:00:00:00:00:16,set:aliyun_google_dns,192.168.1.2,dns-global,2h
dhcp-host=02:00:00:00:00:17,set:aliyun_google_dns,192.168.1.3,dns-local,2h
dhcp-host=02:00:00:00:00:18,192.168.1.250,ap-upstream,2h
```

- 当前 `dnsmasq` 同时下发：
  - 一套动态 GUA
